ハイパーキューブの影響が大変奇妙なかたちでこの場に及んでいるようです。 例えば、 奇妙なゲームの勝者が分かるまで二人のトレジャーハンターの魂が安息に至らないなど。 もしあなたがすでに開始されているゲームの終了を拒んだ場合は何が起こるのでしょうか。

あなたはそれらについて考えることなく取りあえず先へ進むことにしました。 数分もしない内にあなたは見たくなかった本物の最悪の幽霊に出会ってしまいました! 最近訪れた部屋に掛かっていた男の肖像画と似ていることからエッシャー卿自身と判断するのが妥当なようです。 この島から何年もの間トレジャーハンターが帰って来なかった理由が明らかになって来ました。 もはや引き返すことはできません。 あなたは彼の言う通りにするしかなさそうです。

「虚偽の中より真実を告げた者のみを先に進ませる!」 彼があなたを見据えて最初に言った言葉でした。 彼は床にある両面に数が書かれたカードの束を指さしました。 恐らくこれらのカードも過去の訪問者の所持品だったようです。 エッシャー卿は冒険者を試すのにそれを使うことに決めたのでしょうか? しかし今は関係の無いことです。 先へ進むためにあなたは数枚のカードが 1セットに属しているか判断しなくてはなりません。

このミッションでは両面に数字が書かれた特別なカードを使います。 各面に 1つの数字が書かれています。 この数字は以下の決まりで書かれています: 最初のカードは 0 と 1, 2番目は 1 と 2 ... n番目は (n-1) と n。 これらのカードはよくシャッフルされています。

あなたの関数は入力として 1セットの総数 (整数) と数枚のカードの詳細 (整数のリスト) を受け取ります。 各々のカードの片面の数字しか分からないことに注意して下さい。 あなたの仕事はそれらのカードがすべて 1セットに属しているか、他のセットのものが混じっているか判断することです。 以上より True ( カード全てが 1セットから取り出せる場合 ) か False ( それ以外 ) が答となります。

入力: カードの 1セットの総数、選ばれたカードのリスト

出力: 真偽値 ( True か False )
