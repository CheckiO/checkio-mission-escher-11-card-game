ハイパーキューブの影響が大変奇妙な方法でこの場に及んでいるようです。 例えば、 奇妙なゲームの勝者が分かるまで二人のトレジャーハンターの魂が安息に至らないなど。 ふうむ、もしあなたがすでに開始されているゲームの終了を拒んだ場合は何が起こるのでしょうか。

それにもかかわらず、あなたはそれらについて考えることなく先へ進みました。 数分もしない内にあなたは見たくなかった本物の最悪の幽霊に会ってしまいました! 最近訪れた部屋に掛かっていた男の肖像画と似ていることからエッシャー卿自身と判断するのが妥当なようです。 この島から何年もの間トレジャーハンターが帰って来なかった理由が明らかになって来ました。 今や引き返すことはできません。 あなたは彼の言う通りにするしかなさそうです。

「虚偽の中から真実を告げられたものだけを通す!」 彼があなたを見据えて最初に言った言葉でした。 彼は床にある両面に数が書かれたカードの束を指さしました。 恐らくこれらのカードも過去の訪問者の所持品だったようです。 エッシャー卿は冒険者を試すのにそれを使うことに決めたのでしょうか? まあ今は関係の無いことです。 先へ行くためにあなたは最初の数枚のカードが一組のカードに属しているか判断しなくてはなりません。

このミッションでは両面に数字が書かれた特別なカードの束を使います。 各面に一つの数字が書かれています。 この数字は以下の決まりで書かれています: 最初のカードは 0 と 1, 二番目は 1 と 2 ... n番目は (n-1) と n。 これらのカードはよくシャッフルされています。

あなたの関数は入力として一組のカードの総数と数枚のカードを数のリストとして受け取ります。 カードの片面の数字しか分からないことに注意して下さい。 あなたの仕事はそれらのカードがすべて一組のカードの一部か、他のカードの組のものが偶然混じっているか判断することです。 以上より True ( もし見えているカード全てがカード一組から取り出せる場合 ) か False ( それ以外 ) が答となります。

入力: カード組の総数、選ばれたカードのリスト

出力: 真偽値 ( True か False )
